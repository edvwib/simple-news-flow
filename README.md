# simple-news-flow
A simple news flow written in HTML, CSS and PHP

## Installation:
CLone/download the repo, then place the folder with the files in your web root, E.g /htdocs and open index.php in your preferred browser. No need for a database since the data is currently hardcoded into arrays.
